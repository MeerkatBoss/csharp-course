# Консольный калькулятор

Калькулятор для чисел с плавающей точкой, поддерживающий операции сложения, вычитания, умножения
и деления, а также круглые скобки для группировки выражений.

## Запуск
Программу возможно запустить из данной директории (hw1).
```sh
dotnet run
```

## Интерфейс

Программа может вычислять произвольные арифметические выражения, содержащие числа с плавающей
точкой, операции сложения, вычитания, умножения и деления, а также круглые скобки.

Вычисляемое выражение может занимать несколько строк. Для того, чтобы вызвать вычисление введённого
выражения, необходимо ввести символ '='. После символа равенства не допускается наличие символов
кроме пробельных.

Для завершения работы программы необходимо создать ситуацию конца файла (`Ctrl+D` на Linux,
`Ctrl+Z` затем `Enter` на Windows).

Программа также поддерживает запуск с перенаправлением ввода, например:

```sh
echo "(4 + 6) / 2.5 + 8 =" | dotnet run
```
В данном примере программа выведет число 12 и завершит работу.
